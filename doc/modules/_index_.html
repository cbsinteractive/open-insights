Module "index"

Open Insights is a library for building customized RUM clients.

remarks

This is the Open Insights core module. It defines business logic shared by all RUM clients. An Open Insights "provider" specifies the logic particular to a particular RUM service. Each of these is contained within its own package. A "tag owner" builds a RUM client in their own package by importing and utilizing features from the core module and one or more provider modules.

example
import { init, ClientSettingsBuilder } from 'open-insights'
import { Provider, ProviderSettings } from 'open-insights-provider-foo'

const settingsBuilder = new ClientSettingsBuilder()
const fooSettings: ProviderSettings = {
  setting1: 'some value',
  setting2: 'some value,
}

settingsBuilder.addProvider(new Provider(fooSettings))

// Execute a RUM session
init(settingsBuilder.toSettings())
    .then(result => {
        // `result` contains the results from the RUM session after
        // completion
    })

init

  • init(settings: ClientSettings): Promise<SessionResult>

  • Called by tag owner code to initialize a RUM session, either immediately or after some delay.

    remarks

    Waits for the page to load before processing.

    Parameters

    • settings: ClientSettings

      Specifies settings affecting client behavior. These are determined by the tag owner at runtime, so may be used to specify page-level overrides to general defaults.

    Returns Promise<SessionResult>
